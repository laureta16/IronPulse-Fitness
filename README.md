# IronPulse Fitness

![IronPulse Fitness](https://github.com/laureta16/IronPulse-Fitness/actions/workflows/main.yml/badge.svg)

IronPulse Fitness is a modern, responsive, and performance-focused gym website built to showcase a bold, high-energy gym experience. It features sections for strength and conditioning programs, expert trainers, flexible membership pricing, and real member testimonials.

## Features

- **Modern & High-Energy Design**: A sleek user interface that conveys a premium gym experience.
- **Programs Overview**: Details on Strength Training, Cardio Conditioning, Personal Coaching, and Group Classes.
- **Trainers Profile**: Information about expert coaches and their specialties.
- **Pricing Plans**: Clear and concise membership tiers (Basic, Pro, Elite).
- **Testimonials & Reviews**: Real member feedback and a built-in review form.
- **Responsive Layout**: Fully optimized for desktop, tablet, and mobile devices.

## Tech Stack

- **HTML5**: Semantic and accessible markup.
- **CSS3**: Custom styles, modern layout techniques (Flexbox/Grid), and animations.
- **JavaScript**: Interactive features, review handling, and UI dynamics.

## Local Development

If you'd like to run the website locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/laureta16/IronPulse-Fitness.git
   ```
2. Navigate into the project directory:
   ```bash
   cd IronPulse-Fitness
   ```
3. Open `index.html` in your favorite web browser or use a local development server like Live Server (VS Code Extension).
